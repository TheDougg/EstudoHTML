O que é o html??

HTML (Hypertext Markup Language) é uma linguagem de marcação usada para criar a estrutura básica de páginas da web.

Ou seja, não é uma linguagem de programação.

Comandos em HTML:

Índice
  • Texto
  • Links
  • Listas
  • Imagens
  • Formulários

Texto

H1 to H6 - Sãos os títulos (Os textos acima "O que é o html?" e "Comandos em HTML" São H1 e H2 respectivamente.)

P - O comando P significa um parágrafo, ou seja uma linha abaixo dos títulos que são os H's (O texto acima "Comandos em HTML é um H2").

Strong - Deixa o texto em negrito (Assim como o "Strong").

Em - Deixar o texto em Itálico assim como o texto acima "Ou seja, não é uma linguagem de programação"

Links:

A - Este comando faz uma âncora para navegar até outra aba ou tela, resumindo ela faz uma criação de link para quando clicarmos irmos até o link final.

    Obs: utilizar os targets (que significa nada mais nada menos que como iremos querer direcionar a aquele link):

  1. Blank = O link será aberto em uma nova janela ou guia do navegador.
  2. Self = O link ou formulário será aberto na mesma janela ou guia em que o link foi clicado ou o formulário foi enviado. Este é o comportamento padrão se o atributo target não for especificado.
  3. parent = O link ou formulário será aberto na janela pai (a janela que contém o quadro, se o documento estiver em um quadro).
  4. top = O link ou formulário será aberto na janela superior (a janela mais externa, ignorando os quadros).
  5. nome_da_janelaVocê também pode especificar o nome de uma janela específica (por exemplo, aberta anteriormente com JavaScript) como valor para target.

Listas:

ol - Fazer uma lista ordenada, assim como a lista acima com 5 itens em ordem, lembrando que para cada item da lista terá de ser aberto o *li*

ul - Fazer uma lista não ordenada sem ordem 1-2-3-4-5 por exemplo: (Sempre lembrar de utilizar o *li*)

  • teste
  • teste1
  • teste3

li - Item da lista, por exemplo todas essas listas acima eu tive de utilizar o comando li para cada item da lista

Exemplo: *li* teste */li*

Imagens

img - Conseguimos inserir uma imagem no HTML

  1. src = Caminho de onde vem a imagem, ou esta diretamente na pasta do projeto ou o caminho da imagem
  2. alt = Caso a imagem dê error de carregamento, irá mostrar o texto que foi inserido aqui.
